# 🎓 Student Performance Analytics Dashboard

An interactive Student Performance Analytics Dashboard built using Streamlit, Pandas, NumPy, and Matplotlib.

The dashboard helps analyze student academic performance, attendance, GPA trends, department-wise insights, and demographic statistics through interactive visualizations and filters.

---

## 🚀 Features

### 📊 KPI Dashboard
- Total Students
- Average GPA
- Average Attendance
- Total Departments
- Highest GPA
- Average Final Marks
- Pass Rate

### 🎯 Interactive Filters
- Department
- Gender
- Academic Year
- Semester

### 📈 Visualizations
- GPA Distribution
- Attendance Distribution
- Department-wise GPA Analysis
- Gender-wise Performance Analysis

### 📋 Data Tables
- Top 10 Students
- Department Summary
- Filtered Dataset View

### 📥 Export Functionality
- Download filtered data as CSV

## 📊 Dataset Information

Dataset contains 50,000 student records with the following attributes:

| Column Name | Description |
|------------|-------------|
| Student_ID | Unique student identifier |
| Gender | Male/Female |
| Age | Student age |
| Department | Academic department |
| Year | Academic year |
| Attendance | Attendance percentage |
| Quiz_Marks | Quiz scores |
| Assignment_Marks | Assignment scores |
| Midterm_Marks | Midterm examination scores |
| Final_Marks | Final examination scores |
| GPA | Grade Point Average |
| Semester | Semester information |
| Academic_Year | Academic year |

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/student-performance-analytics-dashboard.git
```

Move into the project directory:

```bash
cd student-performance-analytics-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Running the Application

```bash
streamlit run app.py
```

The application will open in your browser at:

```text
http://localhost:8501
```
